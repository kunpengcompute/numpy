ARM64 / Kunpeng performance optimizations
-----------------------------------------

This update includes a series of performance optimizations and validation
improvements for NumPy on ARM64 / Kunpeng platforms, while preserving
compatibility with upstream NumPy behavior.

Highlights include:

* Added and improved ARM64-oriented SIMD optimization paths for selected
  ufuncs, including integer logical operations, shift operations, arithmetic
  operations, floating-point unary operations, and selected reduction paths.
* Added Highway SIMD based optimization paths for selected sort, select,
  argmax, argmin, floor division, pairwise sum, and min/max related operations.
* Improved selected sorting and partition-related paths, including radix sort,
  timsort, sorted-status checking, partition, and argpartition cases.
* Optimized selected histogram, random number generation, indexing, strided
  copy, and mathematical function paths on ARM64 platforms.
* Fixed multiple performance regressions and correctness issues identified
  during benchmark and test validation, including limiting selected
  optimizations to ARM64 paths where appropriate and reverting changes that did
  not provide stable performance gains.

These changes are validated with functional tests and ASV benchmark results.
